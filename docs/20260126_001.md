## 作業概要
1. Spring Initializr でプロジェクトを作成。
2. 必要な依存関係を追加。
3. 基本的なパッケージ構成とクラスを作成。
4. Git リポジトリを初期化し、README.md を追加。
5. Dockerfile と docker-compose.yml を作成（必要に応じて）。
6. CI/CD パイプラインの設定（GitHub Actions など）。
7. ドキュメントを整理し、プロジェクトのセットアップ手順を記載。
8. 初期コミットを作成。
