# APIの新規追加

## 概要
- 1on1管理システムに新しいAPIエンドポイントを追加
- 予定のステータス管理と履歴取得機能を実装

## 追加API仕様
- GET /api/oneonones/{id}/history — 予定の履歴取得
  - 説明: 指定した1on1予定の変更履歴を取得

## 実装状況
- 予定履歴テーブル (`one_on_one_history`) をマイグレーションで追加し、履歴エンティティ/リポジトリ/サービス/DTO を実装済み。
- `GET /api/oneonones/{id}/history` をコントローラに追加し、MockMvc テストでレスポンス内容と 404 ハンドリングを検証済み。
- OpenAPI ドキュメントに履歴エンドポイントとレスポンススキーマを反映済み。
